git webhook php
===============

预打算实现的主要功能：

 + 项目初始化
 + 代码拉取
 + 数据迁移
 + 数据填充


> 运行环境要求PHP5.6以上。
## 参与开发

## 安装
第一种安装方法
```
git clone https://github.com/hardykay/webHoook.git
或者
git clone https://gitee.com/hardykay/webhook.git
```

代码拉取完成后，执行composer安装
```
composer install
```      
第二种安装方法，composer create project
```php               
composer create-project hardykay/webhook hook
```
